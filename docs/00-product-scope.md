# Judgify Product Scope

Judgify is a coding problem platform similar to LeetCode.

The frontend already exists. This repository is backend-focused.

## MVP Features

1. Problem list/detail
2. Admin problem CRUD
3. Auhentication/authorization
4. Test case CRUD
5. Submit code
6. Judge submissions using a custom judge worker
7. Store submission results
8. Return submission status/results to frontend

## Out of Scope for MVP

- Contest
- Ranking
- Discussion
- Editorial
- Plagiarism detection
- Interactive problems
- Payment
- Notification
